# japanese-song-difficulty-analyzer
Estimate the vocabulary and grammar difficulty of Japanese song lyrics for JLPT learners.

## Features

- [ ] Vocabulary analysis
- [ ] Grammar detection
- [ ] Kanji difficulty
- [ ] JLPT recommendation
- [ ] Difficulty visualization

## Tech Stack

- Python
- SudachiPy
- Pandas
- Matplotlib

## Status

Planning

## Note
Due to copyright restrictions, the repository does not include song lyrics. Place your own lyric text files in data/raw/ before running the analysis.
